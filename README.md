# rust-web-server

从零构建的 Rust 量化回测系统，逐步演进为商用交易平台。

---

## 项目现状

### ✅ 已完成

- [x] Actix-Web HTTP 服务器框架
- [x] 事件驱动引擎骨架 (tokio mpsc channel)
- [x] Strategy Trait 可插拔策略抽象
- [x] Candle/K线数据模型 + OHLCV 合法性校验
- [x] CSV 数据解析 (serde 中文别名)
- [x] Portfolio/Position/Order/Audit 数据结构定义

### ❌ 关键缺失

- [ ] Portfolio 从未实际使用 — 引擎不跟踪持仓和现金
- [ ] 订单执行逻辑为空 — OnOrder 只打印日志
- [ ] Audit.profit 硬编码 77.7 — 无真实盈亏计算
- [ ] DefaultStrategy 是占位符 — 固定返回 (price=66.6, amount=100)
- [ ] 仅有 20 条测试数据 — 无法做有意义的回测
- [ ] 无滑点/手续费/印花税模型 — 商用必需
